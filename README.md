# Tic Tac Toe – Low Level Design (Java)

This project implements a TicTacToe game focusing on **Low Level Design (LLD)** principles using Java.

The goal of this project is to demonstrate how a simple game can be designed using **clean architecture, Object Oriented Programming, and design patterns**.

## Key Concepts

- Object Oriented Programming (OOP)
- Low Level System Design
- Strategy Design Pattern
- Modular and Scalable Architecture
- Separation of Concerns

## Core Components

GameController – Handles the game flow  
Game – Maintains game state  
Board – Represents the game board  
Cell – Represents each board position  
Player – Stores player information  
Move – Tracks player moves  

## Winning Strategy

The project uses the **Strategy Design Pattern** to implement different ways of determining the winner.

Strategies implemented:

- Row Winning Strategy
- Column Winning Strategy
- Diagonal Winning Strategy

This design allows easy extension for new winning rules without modifying existing code.
